# StoreApp

## Описание
StoreApp е уеб приложение тип онлайн магазин, разработено с ASP.NET MVC.
Потребителите могат да разглеждат и търсят продукти, а администраторът може да ги управлява.

## Проблем
Малките онлайн търговци често нямат лесна и достъпна платформа за представяне и управление на продукти.

## Функционалности
- Регистрация и вход в системата
- Роли: администратор, потребител и гост
- Добавяне, редактиране и изтриване на продукти (администратор)
- Преглед на продукти
- Търсене на продукти
- Потвърждение при изтриване

## Използвани технологии
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- HTML, CSS

## Стартиране
1. Клонирай проекта от GitHub
2. Отвори проекта във Visual Studio
3. Настрой връзката към базата данни в `appsettings.json`
4. Стартирай проекта с IIS Express или `dotnet run`
